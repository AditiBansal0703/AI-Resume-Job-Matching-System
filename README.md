- Upload resume (PDF)

- Compare with job description
- Generate match score and suggestions
- Frontend + FastAPI backend

##  Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** FastAPI (Python)
- **Libraries:** scikit-learn
